# Simulator

To demonstrate behavior of OBU and RSU without actual sensor, this program will simulate with fork process.

## Setup Guide (First time only)

### Prerequisite

setup `OBU` and `RSU` before

### Step

1. Go to `server` dir
2. `npm i`
3. Repeatly, with `web` dir

## Start

run

```
npm run start:dev
```

in `server` dir
